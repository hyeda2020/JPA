# 3. 연관관계 매핑 기초
- 단방향 연관관계
  ```
  public class Team {
      @Id
      @GeneratedValue
      @Column(name = "team_id")
      private Long teamId;
  }
  
  public class Member {
      ...
      /* Team 클래스와 단방향 연관관계 매핑(다대일) */
      @ManyToOne
      @JoinColumn(name = "team_id")
      private Team team;
  }
  ```
